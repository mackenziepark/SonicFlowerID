## Sonic Flower ID

Sonic Flower ID is a personalized digital identity generation system that creates a unique flower based on a user's name, birthday, and voice characteristics.

## Project Overview

This project explores the concept of representing personal identity through generative visual design.

User information and voice data are analyzed to generate a unique flower pattern that serves as a digital identity.

## Features

- User name and birthday input
- Voice recording and voice feature extraction
- SHA-256 based identity generation
- Personalized flower generation
- Animated flower visualization
- Digital ID card generation

## Project Structure

### FlowerIDAdvanced.java
Java Swing prototype implementation.

Main technologies:
- Java Swing (GUI)
- Audio Processing
- FFT (Fast Fourier Transform)
- SHA-256 Hashing
- Procedural Flower Generation

### WebVersion
Web-based implementation of the same concept.

Main technologies:
- HTML
- CSS
- JavaScript

The web version recreates the original Java application in a browser environment and provides an interactive user experience for generating personalized flower identities.

## Concept

Each user receives a unique flower identity generated from:

- Name
- Birthday
- Voice volume
- Voice frequency characteristics

The generated flower acts as a visual representation of personal identity.
